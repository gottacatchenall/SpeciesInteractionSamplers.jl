
# Getting Started with `SpeciesInteractionSamplers.jl` {#Getting-Started-with-SpeciesInteractionSamplers.jl}

Load the package.

```julia
using SpeciesInteractionSamplers
using CairoMakie
```


Generate a network

```julia
net = generate(NicheModel())
```


```ansi

```


Make it realizable, first by generating a relative abundance distribution

```julia
julia> relabd = generate(NormalizedLogNormal(), net)
```
