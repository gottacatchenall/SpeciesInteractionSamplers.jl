# SpeciesInteractionSamplers.NicheModel — Type.

NicheModel

A NetworkGenerator that generates a food-web for a fixed number of species and an expected value of connectance (the proportion of possible edges that are feasible in the network). Proposed in @Williams2000SimRul.

# SpeciesInteractionSamplers.SpeciesPool — Type.
julia
SpeciesPool{S}

A SpeciesPool represents a set of species, each identified with either a Symbol or String in a vector called names. Note that the order of species in names is assumed to be fixed, so the index of each species in names can be used an an integer identifier.

source

# SpeciesInteractionSamplers.StochasticBlockModel — Type.
julia
StochasticBlockModel <: NetworkGenerator

A NetworkGenerator that generates a network based on assigning each node a group node_ids, and a matrix mixing_matrix which at each index mixing_matrix[i,j] describes the probability that an edge exists between a node in group i and a node in group j.

source
